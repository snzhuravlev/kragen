Real-time output from the task worker (SSE). Lines prefixed with [kragen] are gateway progress (load history, memory, spawn CLI, finish). The Cursor agent's own stdout/stderr follow; then the stream ends when the task is fully saved. Shown as Markdown while the run is in progress. By default the browser does not abort the stream (long tasks are OK). Optional: ?streamTimeoutMs=180000 to cap the wait (milliseconds).

Logical workspace file tree backed by object storage. Uploads here are visible to Telegram uploads saved under /Inbox/Telegram.
